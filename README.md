# 口罩地圖-練習

## 使用模組
| 模組名稱      | 功用          |
| ------------- |:-------------:|
| axios         | http請求      |
| vue-axios     | http請求      |
| element-plus  | ui庫          |
| leaflet       | 地圖應用框架  |
  
## 編成心得
>這個練習題中我使用了Vue3框架搭配element-plus ui庫，<br>
使用 vue-axios 取得藥局數據，與 leaflet 做串接，<br>
使用 typescript 製作了 藥局資了結構的 interface， <br>
來做引數及變數的註解
